# Bosnia Connect

An advanced cross-platform social network and messenger application designed to connect users with rich features including AI integration, messaging, feeds, profiles, and more.

## Features
- **Customizable Auth:** Google Auth v2.0 UI for easy login and signup.
- **Global Search:** Accessible directly from the loading/splash screen.
- **Advanced Messenger:** Send photos, videos, and use AI grammar correction for multiple languages (Bosnian, Croatian, Serbian, English, German, Spanish, Italian). WebRTC ready mockups for call/video features.
- **Rich Feed:** Post news, readable HTML links, embedded YouTube videos (including shorts), and emoticons. Features an AI writer for content creation.
- **Meta AI Assistant:** Free photo/video generation prompts, Q&A, and grammar correction features integrated across the app.
- **Verified Profiles:** Support for User, Page, and Group profiles with customizable photos/covers and automatic Free, Gold, and Blue verification badges.
- **Leaderboard & XP:** Gamification with profile XP and ranking.
- **Notifications:** Comprehensive UI with structural support for sound integration.

## Tech Stack
- Flutter (iOS, Android, Web, Desktop)
- Supabase (Backend structure)
- Material Design 3

## Getting Started

1. Ensure you have Flutter installed (SDK ^3.7.2).
2. Clone the repository and run `flutter pub get`.
3. Run the app using `flutter run`.
